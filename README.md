Music Playlist Analyzer<br>
A Python-based data analysis tool designed to process music datasets, calculate track statistics, and visualize listening preferences using industry-standard libraries.<br>
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Overview<br>
This repository contains a script, music_player.py, which automates the analysis of a music library stored in CSV format.<br> 
The tool provides both a programmatic summary of the data and a visual breakdown of song durations and genre distributions.<br>
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Technical Stack<br>
Language: Python<br>
Data Manipulation: pandas<br>
Data Visualization: matplotlib<br>
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Key Features<br>
Automated Statistical Analysis: Calculates the mean duration of tracks and identifies the most frequent genre (mode) within the dataset.<br>
Dynamic Labeling: Includes rotated x-axis labels and customized titles for professional reporting.<br>
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Visualizations<br>
The script generates two primary figures to represent the data:<br>
Duration per Song (Bar Chart): A comparison of individual tracks against their duration in seconds.<br>
Genre Distribution (Pie Chart): A percentage-based breakdown of the playlistâ€™s genre composition<br>
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Data Insights<br>
Based on the analysis of the sample dataset, the following trends were identified:<br>
Genre Preference: The playlist is primarily composed of Pop music.<br>
Track Length: Listening habits lean toward shorter tracks on average.<br>
Era focus: The collection consists largely of modern music from recent years.<br>
